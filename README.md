# Winter-2026-CS-ECE-599-labs

Labs for **CS/ECE 599 (Winter 2026)** — Advanced Topics in Deep Learning & High-performance Computing.

## Repo structure
- `labs/lab0` … `labs/lab5`: lab instructions + handouts
- `docs/`: setup guides (local + OSU COE HPC)
- `shared/`: shared utilities (env files, SLURM templates)
- `scripts/`: helper scripts (setup, packaging, etc.)

2. HPC + VS Code setup guide: `docs/hpc-vscode.md`
